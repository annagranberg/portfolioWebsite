Wedding website

I created this wedding website (in Swedish) using HTML, CSS, JavaScript and Node.js. It includes an API-powered gallery, RSVP functionality, interactive sections and a smooth, modern design for big and small screens. This was a fun project that allowed me to combine my technical skills with creativity to build a personalized and user-friendly website for our own wedding.

Virtual Plant

This is a virtual plant project my project group and I created using Java. It’s a fun and interactive game where you can take care of your own virtual plant. You can water it and watch it grow over time. The plant has different stages of growth and you can see how it evolves based on your care.
